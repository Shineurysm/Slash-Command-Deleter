## Prerequisites

- npm >=5.5.0
- node >=9.3.0
- discord.js >= 13.8.1
- dotenv >= 16.0.1
- prompt-sync >= 4.2.0
- cli-color >= 2.0.2

## Setup

```
Create an '.env' file and copy and paste the text below into your '.env' file

TOKEN=YOUR_BOT_TOKEN
```

## Install All Required Prerequisites

```sh
npm install
```

